## 📦 资产更新日志（未混淆目录）
### v3.4.3
资产文件: 

＃⚙️错误修复和改进

 - 添加了NAT64作为代理IP的替代方法。您可以使用名为`nat64_prefix`＃932＃977的环境变量添加一些Nat64前缀或通过CloudFlare仪表板添加。
 - 改进的面板UI和数据输入验证
 - 将默认Vless/Trojan指纹设置为“ chrome”
 - 一些错误修复

> [！小心]
>您必须在升级到此版本后更新订阅，新部署无需做任何事情。

> [！警告]
>在免费的公共NAT64前缀中，只有少数与面板兼容。首先测试面板的默认NAT64前缀。如果有效，您可以尝试其他前缀以查看哪些有效。

> [！提示]
> NAT64前缀格式为[IPv6]，没有CIDR符号，因此，如果您发现了2001：67C：67C：2960：6464 ::/96的Nat64，则应将其转换为[2001：67C：2960：6464：6464 ::]。

> [！提示]
> nat64在CloudFlare仪表板环境中的前缀输入格式变量`nat64_prefix`与`prodxy_ip`的相同，即IPS应该以逗号分隔填充IPS，与应填充每个的面板相反。
